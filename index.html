Revision tree visualizer

Pro tip

You can you this tool to create any revision tree you want and then export it to JSON. Click on nodes to open edit form. Edit key/value by double-clicking on it. Keep in mind that _rev property does not get changed automatically so you usually want to change 3-abcd to something like 4-klmn. Open console and type exportDoc() to export document into bulkDocs friendly result.
